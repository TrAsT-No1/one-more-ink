# One More Ink — Concept Website

## What This Is

Sito web speculativo per One More Ink, tattoo & piercing studio a Modena (Via Nobili 20/22) attivo da 7+ anni senza presenza web. Un sito dark, premium, con animazioni scroll e portfolio per artista — costruito come pezzo forte del portfolio di Federico Bavieri (TrAsT No1) e come proposta diretta al cliente.

## Core Value

Un sito tattoo di livello Awwwards che il cliente non può rifiutare e che dimostra le capacità di Federico a qualsiasi prospect futuro.

### Strategia di approccio
Il sito viene costruito completo e deployato. Federico lo manda via DM a Federica (titolare) con un messaggio tipo "ho fatto questo per voi, se vi piace parliamo". Il prezzo target è €1.500.

## Constraints

- **Tech stack**: Next.js 15 + Tailwind CSS v4 + Framer Motion — coerente con le competenze di Federico
- **Immagini**: Placeholder/mock — non abbiamo foto reali dello studio o dei lavori. Useremo immagini di riferimento da Instagram o placeholder con istruzioni su dove inserire le foto reali
- **Deploy**: Vercel (free tier per il concept)
- **Timeline**: Concept completo e navigabile, qualità > velocità
- **Budget**: Zero (è speculativo, l'investimento è il tempo di Federico)
