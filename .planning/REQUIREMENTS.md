# Requirements — One More Ink

## v1 Requirements

### Portfolio & Artists
- [ ] **PORT-01**: Visitor can browse artist portfolio gallery with lazy-loaded, high-quality images per artist (Federica Morselli, Stefano Mazzotta)
- [ ] **PORT-02**: Visitor can filter portfolio images by tattoo style (traditional, blackwork, fine line, ornamental, neo-traditional, realismo)
- [ ] **PORT-03**: Visitor can open full-screen lightbox view of any portfolio image
- [ ] **PORT-04**: Each artist has dedicated page with bio, photo, specialties, and personality description
- [ ] **PORT-05**: Visitor can view flash sheet gallery with available/taken status and price/size info

### Walk-In Day
- [ ] **WALK-01**: Dedicated Walk-In Day event page with date, flash designs, how-it-works section, and booking CTA
- [ ] **WALK-02**: Event page shows countdown timer to next Walk-In Day
- [ ] **WALK-03**: Flash designs on Walk-In Day page link to flash sheet gallery (PORT-05)

### Contact & Booking
- [ ] **BOOK-01**: WhatsApp CTA button visible on every page (sticky header or floating button)
- [ ] **BOOK-02**: Instagram DM link as secondary booking CTA on every page
- [ ] **BOOK-03**: Contact section with phone number, WhatsApp link, Instagram link, and studio address

### Studio Info
- [ ] **INFO-01**: Studio info page with embedded Google Maps, address (Via Nobili 20/22, Modena), parking info, and directions
- [ ] **INFO-02**: Opening hours displayed clearly (Mar-Ven 13:00-19:30, Sab 10:30-18:00)
- [ ] **INFO-03**: Aftercare instructions page (static content, clean typography)
- [ ] **INFO-04**: FAQ section with accordion/collapse format covering: first tattoo, pricing method, session length, what to bring, healing, deposits

### Social Proof
- [ ] **SOCIAL-01**: Testimonials section with curated client quotes (first name + tattoo style)
- [ ] **SOCIAL-02**: Instagram link prominently displayed with @onemoreink handle
- [ ] **SOCIAL-03**: Instagram feed embed or recent posts preview (progressive: link at launch, feed later)

### Design & UX
- [ ] **UX-01**: Dark editorial design as default theme (black backgrounds, gold/warm accents, high contrast) — not a toggle, it IS the brand
- [ ] **UX-02**: Mobile-first responsive design optimized for Instagram link-in-bio traffic (80%+ mobile)
- [ ] **UX-03**: Scroll-triggered entrance animations, parallax on hero images, hover effects on gallery
- [ ] **UX-04**: Navigation flat and minimal: Home, Artists (Federica, Stefano), Gallery, Walk-In Days, Info, Contact (max 6 items)
- [ ] **UX-05**: Hero section with dark, cinematic aesthetic and animation d'impatto

### Performance & SEO
- [ ] **PERF-01**: Largest Contentful Paint < 2 seconds on mobile
- [ ] **PERF-02**: Images optimized (WebP/AVIF, srcset, lazy loading) for image-heavy portfolio
- [ ] **PERF-03**: Lighthouse score 90+ on all metrics
- [ ] **SEO-01**: Meta tags, Open Graph, and semantic HTML on all pages
- [ ] **SEO-02**: JSON-LD LocalBusiness structured data (name, address, hours, phone, geo coordinates)
- [ ] **SEO-03**: Alt text on all images, Italian-language content optimized for "tatuatore Modena" / "tattoo studio Modena"

## v2 Requirements (Deferred)

- [ ] **MERCH-01**: Merch catalog page with t-shirt photos and "DM to order" Instagram link
- [ ] **FORM-01**: Multi-step consultation request form (style, placement, size, reference images, budget) with artist auto-routing
- [ ] **BLOG-01**: Blog/knowledge section with cornerstone articles (aftercare deep-dive, style guide, first tattoo guide)
- [ ] **I18N-01**: English translation of key pages for international visitors
- [ ] **GUEST-01**: Guest artist announcement cards (artist, dates, style, booking link)

## Out of Scope

- Full booking/calendar system — WhatsApp workflow works for 2-artist studio (A1)
- Background music / auto-play audio — instant bounce (A2)
- Complex navigation / mega menus — max 6 pages (A3)
- Stock photography — only real tattoo/studio photos (A4)
- Price list / pricing calculator — bespoke pricing, consultation-based (A5)
- Client accounts / login — no auth needed (A6)
- AI tattoo generator — undermines artist-led value (A7)
- Heavy 3D / WebGL — kills mobile performance, no ROI (A8)
- Chat widget / chatbot — WhatsApp covers this (A9)
- Newsletter signup — Instagram is the notification channel (A10)
